## 2. 가변 인자 리스트

가변 인자 리스트를 이용하여 입력받은 정수들의 평균값을 반환하는 함수

```python
def my_age(*lst):
    sum = 0     
    for i in lst:
        sum += i
    return sum / len(lst)
```

```
1. 가변 인자를 리스트를 통해 prameter로 받는다.
2. for문으로 리스트의 인자를 하나씩 받아 합한다.
3. len()함수를 사용하여 리스트의 개수를 구해서 평균값으로 반환한다.
```

## 4.  이름 공간

python에서 변수를 찾을 때 접근하는 이름 공간을 순서대로 작성.

```python
총 4가지의 scope가 존재한다
1. local scope                : 현재 작업 중인 범위
2. Enclosed scope (기억안남)   : local 한 단계 위 범위
3. global scope               : 최상단에 위치한 범위
4. built-in scope             : 모든 것을 담고 있는 범위
```
